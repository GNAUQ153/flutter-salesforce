# Flutter Salesforce Integration

A demo project that integrates a **Flutter mobile app** with **Salesforce** through a **Node.js backend**.  
The system supports Salesforce login, fee retrieval, and fee payment through backend APIs.

## Project Overview

This project is built for learning and demonstration purposes, focusing on:

- Flutter app development
- Salesforce OAuth integration
- Backend API development with Node.js
- Communication between mobile app and Salesforce services

## Project Structure

```bash
flutter-salesforce/
├── flutter_app/      # Flutter mobile application
├── sf-backend/       # Node.js backend for Salesforce integration
├── .gitignore
└── README.md
```
